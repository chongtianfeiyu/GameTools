Classpublic class CanvasButtonAccordionHeader
InheritanceCanvasButtonAccordionHeader CanvasButton mx.controls.Button
Implements mx.core.IDataRenderer

The CanvasButtonAccordionHeader class allows you to easily use a CanvasButton control as the header renderer of an Accordion. This class is a copy of the AccordionHeader class, but instead of subclassing Button, it subclasses CanvasButton.
